Based on Bootstrap
=============================================
WordPress Plugin v. 0.95
To change color: Change variablees (main_color, main_color_hover) in styles.less file and compile less to css (styles.css)
To make right colors go to: http://www.google.com/design/spec/style/color.html and select 1 palette. Value of 700 is base color (main color variable) and A700 is second color (main_color_hover)
